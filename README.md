# 📞 Contact Management System (CLI)

## 📌 Project Description
This is a Python-based Command Line Interface (CLI) Contact Management System.  
It allows users to store, manage, search, and filter contacts efficiently.

---

## 🚀 Features
- Add new contacts
- View all contacts
- Search contacts (by name, phone, email)
- Filter contacts (by city and company)
- Update contact details
- Delete contacts
- Data stored in JSON file

---

## 🛠 Technologies Used
- Python
- JSON (for data storage)

---

## ▶️ How to Run
1. Install Python
2. Download project
3. Open terminal in project folder
4. Run:

```bash
python main.py
